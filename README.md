🧪 Разработка низкомодульного 1К ПУ строительного герметика
Data-driven подход к формулированию высокоэластичного однокомпонентного полиуретанового герметика влажностного отверждения для фасадных швов.

Проект представляет собой замкнутый цикл R&D: от автоматизированного парсинга технических спецификаций (TDS) конкурентов с использованием LLM до математического моделирования стехиометрии и генерации плана лабораторных экспериментов (DOE).

🎯 Техническое задание (Target Profile)
Твердость по Шору А: 20 – 25 (низкомодульный шов для высокой деформационной нагрузки)
Относительное удлинение при разрыве: > 600%
Время образования поверхностной пленки (Skin time): 40 – 70 минут (при 23°C, 50% RH)
Экологические ограничения: Полный отказ от фталатных пластификаторов (использование DINCH/бензоатов).

🚀 Ключевые инженерные решения
1. Каскадный NLP/LLM Парсер (src/data_collection/)
Для извлечения структурированных метрик из неструктурированных PDF реализован 4-этапный каскад с отказоустойчивостью:

Regex (Lines & Sliding Window) — точный поиск по шаблонам.
Table Extraction (pdfplumber) — парсинг ячеек таблиц технических свойств.
LLM Fallback (OpenRouter/Groq) — отправка сырого текста в Llama-3 для извлечения скрытых или нестандартно оформленных данных с жестким JSON-schema enforcement.
Physical Validator — автоматический отсев галлюцинаций LLM через проверку на физическую адекватность (например, сброс Elongation < 100% для эластомеров).
2. Physics-Informed Симулятор рецептур
Вместо слепого машинного обучения используется суррогатное моделирование на основе физико-химических законов:

Связь молекулярной массы полиола ($M_n$) и индекса $[NCO]/[OH]$ с плотностью сшивки.
Влияние объема пластификации и наполнителя ($CaCO_3$) на твердость и кинетику отверждения.
3. Автоматизация DOE (Design of Experiments)
Для лабораторной верификации математической модели генерируется матрица Бокса-Бенкина (Box-Behnken Design). Это позволяет аппроксимировать поверхность отклика второго порядка, сократив количество необходимых синтезов с 27 (полный факторный эксперимент) до 15.
