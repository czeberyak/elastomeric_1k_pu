# Разработка низкомодульного 1К ПУ строительного герметика

Проект по созданию высокоэластичного однокомпонентного полиуретанового герметика влажностного отверждения для фасадных швов по техническому заданию.

## 🎯 Техническое задание и целевые метрики (Targets)
* **Твердость по Шору А:** 20 – 25
* **Относительное удлинение при разрыве:** > 600%
* **Время образования поверхностной пленки (Skin time):** 40 – 70 минут (при 23°C, 50% RH)
* **Экологические ограничения:** Полный отказ от фталатных пластификаторов.

## 🏗 Архитектура проекта и репозитория
Проект построен по методологии Data Science Pipe (Cookiecutter-подобная структура):
* `data/01_raw/` — технические паспорта аналогов (TDS) в формате PDF.
* `data/02_interim/` — десериализованный сырой текст документов.
* `data/03_processed/` — структурированные матрицы признаков (CSV).
* `src/data_collection/` — автоматизированные парсеры текстовых массивов.
* `src/features/` — математические модули расчета стехиометрии и ПУ-индексов ($NCO/OH$).

## 💻 Стек технологий
* **Окружение:** VS Code, WSL2 (Ubuntu), Python 3.10+, Virtualenv
* **Анализ данных & ML:** Pandas, Numpy, Matplotlib, Seaborn, Scikit-learn
* **Инженерия данных:** Pdfplumber, Regular Expressions (Regex)

## 📊 Текущий статус проекта
1. Реализован и оттестирован модуль автоматического сбора данных с TDS конкурентов (`parser_tds.py`).
2. Проведен разведочный анализ рынка (EDA) в Jupyter Notebook, определены референтные бенчмарки (SikaHyflex-250 Facade).
3. Разработан стехиометрический калькулятор `pu_calculator.py` для генерации сетки рецептур NCO-форполимеров на основе высокомолекулярных полиэфирдиолов (ППГ-2000/3000/4000) и 4,4'-MDI.
